## Casos de Uso para MVP

#### 1. Como requisitante eu quero poder fazer login no sistema;

#### 2. Como requisitante eu quero poder cadastrar um modelo de proposta;

#### 3. Como requisitante eu quero poder cadastrar uma proposta;

#### 4. Como requisitante eu quero poder gerar um link para cada fornecedora, que possibilitará o cadastro de uma única proposta;

#### 5. Como requisitante eu quero poder verificar se a proposta já foi cadastrada;

#### 6. Como requisitante eu quero poder editar uma proposta;

#### 7. Como requisitante eu quero poder excluir uma proposta;

#### 8. Como fornecedora eu quero poder cadastrar uma proposta por meio de link gerado pelo requisitante.
